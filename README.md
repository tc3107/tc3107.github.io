# GitHub Pages IT/Cybersecurity Portfolio

Static portfolio for an entry-level IT and cybersecurity candidate. The site uses plain HTML and CSS only, with no build step or framework dependency.

## Pages

- `index.html` - Home
- `projects.html` - Project overview
- `cv.html` - CV
- `contact.html` - Contact
- `projects/linux-infrastructure.html` - Linux infrastructure case study
- `projects/self-hosted-lab.html` - Self-hosted lab case study
- `projects/linux-hardening.html` - Linux hardening case study

## Deploy on GitHub Pages

1. Push these files to the repository main branch.
2. In GitHub, open Settings, then Pages.
3. Set the source to deploy from the main branch root.
4. Save the setting and wait for GitHub Pages to publish the site.

## Customization

Update the name, email, GitHub URL, project details, and CV content directly in the HTML files.
